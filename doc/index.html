KIX 18 - Frontend

Architecture

Intruduction

The KIX18 frontend contains a application framework, web server (REST-API) and web application in once. The frontend server (based on NodeJS) provides a web server instance (express) for http communication. Over the http default route (http://<fqdn>:<port>/) the web server delivers the basic web application. The web application itself is a single page application (SPA). For data requests and responses it uses sockets (Socket.io) to communicate with the frontend server.

Layers

As described in the introduction the KIX18 frontend is a bundle of different services and workers.
